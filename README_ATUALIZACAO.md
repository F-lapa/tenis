# Atualização profissional

Substitua o `index.html` atual e envie também:

- manifest.webmanifest
- service-worker.js
- pasta icons
- pasta downloads

## Instalação no celular

O botão **Instalar aplicativo** usa PWA e funciona no Chrome Android após a publicação no GitHub Pages.

## APK

Um APK real precisa ser compilado e assinado como aplicativo Android. Para ativar o botão **Baixar APK**, coloque o arquivo pronto em:

`downloads/fernando-lapa.apk`

O sistema verifica automaticamente se o arquivo existe antes de liberar o botão.
